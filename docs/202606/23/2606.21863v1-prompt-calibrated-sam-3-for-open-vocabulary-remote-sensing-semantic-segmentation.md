## 摘要
遥感图像中的开放词汇语义分割旨在分割超出固定标签空间类别的对象。近期基于SAM 3的方法提供了有前景的无需训练基础，但仍存在三个关键问题：（1）单一类别名称提示对复杂遥感类别的语义覆盖不足；（2）将每个类别扩展为多个提示会引入冗余的在线文本编码；（3）直接聚合多个提示响应会将噪声激活传播到最终预测中。为解决这些问题，我们提出ProC-SAM3，从三个互补方面校准SAM 3的提示接口，以用于遥感开放词汇语义分割。首先，我们构建一个离线提示池，其中类别匹配器将MLLM生成的候选提示分组到每个类别集合中，并通过扩展约束利用类别先验知识进一步细化每个集合。其次，生成的文本嵌入被缓存并在所有测试图像中重用，从而消除重复的文本编码。第三，我们引入存在引导残差融合，通过提示存在性和置信度对不可靠的解码器输出进行门控，随后进行峰值保持类别聚合，以保留小目标和稀疏目标的细粒度激活。在八个基准上的实验表明，ProC-SAM3的平均mIoU达到56.1%，比先前最佳的无训练方法高出3.9个百分点。代码将在https://github.com/YanghuiSong/ProC-SAM3提供。

## Abstract
Open-vocabulary semantic segmentation (OVSS) in remote sensing images aims to segment categories beyond a fixed label space. Recent SAM 3-based methods provide a promising training-free foundation, yet three key issues remain: (1) a single class-name prompt lacks sufficient semantic coverage for complex remote sensing categories; (2) expanding each category into multiple prompts introduces redundant online text encoding; and (3) directly aggregating multiple prompt responses propagates noisy activations into the final prediction. To address these issues, we propose ProC-SAM3, which calibrates SAM 3's prompt interface for remote sensing OVSS from three complementary aspects. First, we construct an offline prompt pool where a Category Matcher groups MLLM-generated candidates into per-category sets, and Expansion Constraints further refine each set using category-specific prior knowledge. Second, the resulting text embeddings are cached and reused across all test images, eliminating repeated text encoding. Third, we introduce Presence-Guided Residual Fusion to gate unreliable decoder outputs by prompt presence and confidence, followed by peak-preserving class aggregation that retains fine-grained activations for small and sparse objects. Experiments on eight benchmarks show that ProC-SAM3 achieves an average mIoU of 56.1%, outperforming the previous best training-free method by 3.9 percentage points. Code will be available at https://github.com/YanghuiSong/ProC-SAM3.

#### 1. 核心问题与整体含义（研究动机和背景）
- **问题**：遥感图像开放词汇语义分割（OVSS）旨在分割超出固定标签集的任意类别。现有基于SAM 3的无需训练方法（如SegEarth-OV3、ConceptBank）虽然提供了强大基础，但存在三个关键缺陷：
  1. 单一类别名称提示语义覆盖不足，无法应对复杂遥感类别（如“车”可能包含小汽车、大货车等不同形态）。
  2. 若将一个类别扩展为多个提示，会带来大量冗余在线文本编码开销。
  3. 直接聚合多个提示的响应会将噪声激活（如误检区域）传播到最终预测中，尤其会抑制小目标的细粒度激活。
- **动机**：提出一个全训练-free、无需标注的框架，从提示构建、编码效率和聚合策略三个层面系统地校准SAM 3的提示接口，以提升遥感OVSS性能。

#### 2. 方法论：核心思想、关键技术细节
- **整体思路**：ProC-SAM3 包含三个阶段（如图2所示）：
  - **离线提示池构建（Offline Prompt-pool Construction, OPC）**：
    - 使用多模态大模型（Qwen3-VL-8B）为每个基础类别生成多样化候选描述，再通过**类别匹配器**（Category Matcher）将候选映射到正确类别（如“小型车辆”“大型车辆”归入“车辆”类）。
    - 应用**扩展约束**（如属性、描述、场景等）过滤语义模糊或视觉不合理的候选，生成每个类别的紧凑提示集。所有类别提示合并为全局离线提示池 \( P^* \)，每个提示 \( q \) 带有类别标签 \( \pi(q) \)。
  - **提示优化批处理推理（Prompt-Optimized Batched Inference, POBI）**：
    - 将离线提示池一次性通过SAM 3文本编码器，缓存为持久化文本嵌入 \( T\{Q\} \)。推理时对所有测试图像重复使用，避免重复编码。
    - 图像批次通过视觉编码器得到共享视觉特征 \( F \)，然后与缓存的文本嵌入分块（batch size B）送入掩码解码器，输出每个提示的语义掩码 logits \( S_q^{sem} \)、实例掩码 logits \( S_q^{ins} \)、存在分数 \( \rho_q \) 和实例置信度 \( k_q \)。
  - **存在引导残差融合（Presence-Guided Residual Fusion, PGRF）**：
    - 计算**一致性** \( A_q = 1 - |S_q^{sem} - S_q^{base}| \)（\( S_q^{base} = \max(S_q^{sem}, S_q^{ins}) \)）。
    - 计算**正残差** \( \Delta_q = [S_q^{base} - S_q^{sem}]_+ \)。
    - 门控因子 \( G_q = \rho_q \cdot k_q \cdot A_q \)。
    - 融合后响应：\( \hat{S}_q = \rho_q \cdot (S_q^{sem} + G_q \odot \Delta_q) \)。
    - 最后通过像素级最大池化跨类别聚合：\( \tilde{S}_i(u) = \max_{q:\pi(q)=i} \hat{S}_q(u) \)，取 argmax 得到最终预测图。
