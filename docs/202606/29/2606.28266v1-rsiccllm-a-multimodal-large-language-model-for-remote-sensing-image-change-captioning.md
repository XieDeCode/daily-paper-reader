## 摘要
遥感图像变化描述旨在描述双时相遥感图像之间的变化，具有重要的研究和应用价值。然而，现有方法大多依赖传统深度学习架构，有限的模型能力制约了性能。尽管大模型后训练技术在通用领域取得了巨大成功，但由于数据稀缺和细粒度变化理解的需求，其直接迁移到遥感图像变化描述仍面临挑战。为此，我们提出了RSICCLLM，这是首个用于遥感图像变化描述的大视觉-语言模型后训练框架。具体而言，我们设计了一种数据生成范式，发布了指令数据集RSICI，并建立了任务特定的遥感图像变化描述基准。我们进一步引入差异感知有监督微调，以显式提取变化表示并引导模型感知和理解时序差异。此外，我们提出了双负偏好优化，采用两种互补的负样本构建策略来构建偏好数据集RSICP，并进一步优化模型性能。大量实验验证了RSICCLLM的优越能力，其仅凭7B参数即取得了显著优于更大规模模型的结果。代码和数据集将在https://github.com/keaill/RSICCLLM公开。

## Abstract
Remote Sensing Image Change Captioning (RSICC) aims to describe changes between bi-temporal remote sensing images and holds significant research and application value. However, most existing methods rely on conventional deep learning architectures, and the limited model capacity constrains performance. Although large-model post-training techniques have achieved great success in general domains, their direct transfer to RSICC remains challenging due to data scarcity and the need for fine-grained change understanding. To address this, we propose RSICCLLM, the first post-training framework for large vision-language models in RSICC. Specifically, we design a data generation paradigm, release the instruction dataset RSICI, and establish a task-specific RSICC benchmark. We further introduce Difference-aware Supervised Fine-tuning to explicitly extract change representations and guide the model in perceiving and understanding temporal differences. In addition, we propose Dual-Negative Preference Optimization (DNPO), which employs two complementary negative-sample construction strategies to construct the preference dataset RSICP and further refine model performance. Extensive experiments validate the superior capability of RSICCLLM, which achieves outstanding results with only 7B parameters, surpassing models of substantially larger scales. The code and dataset will be made publicly available at https://github.com/keaill/RSICCLLM.

## 2. 方法论：核心思想、关键技术细节

### 总体框架：RSICCLLM，三阶段流水线

#### 阶段1：指令数据生成（Instruction Data Generation）
- **数据源**：整合LEVIR-CD、SYSU-CD（约35k样本，32k训练+3k域内测试）及S2Looking、CDD（约5k域外测试）。
- **生成方式**：使用Qwen-VL-Max多模态大模型，结合二进制变化mask作为几何先验，生成细粒度变化描述。
- **质量控制**：设置规则过滤（关键词检测、长度限制、语义幻觉检查），人工专家复审，同义词替换增强语言多样性。
- **产出**：RSICI（遥感图像变化指令数据集），约40k图像、20k说明（平均77词），是目前最大的通用域RSICC指令集。

#### 阶段2：差异感知监督微调（Difference-aware Supervised Fine-tuning）
- **基础模型**：Qwen2.5-VL-7B。
- **显式变化提取**：
  - 计算初步差异图 `T = I1 - I2`（token化后）。
  - 引入8种几何敏感卷积核（Central Difference Convolution + 方向性不对称/对称算子），对双时相图像分别提取纹理特征。
  - 对每个特征图执行Hough变换（参数空间投票），再通过逆Hough变换恢复结构响应，避免早期差分导致语义丢失。
  - 计算归一化几何-纹理特征差异 `ΔS = (S(2) - S(1)) / (||S(1)||1 + ||S(2)||1 + ε)`。
- **融合模块**：使用交叉注意力（cross-attention）以T为query、ΔS为key/value，融合得到最终变化表示F。
- **增强视觉嵌入**：`V_enhanced = V_orig + F`，使模型更关注变化区域。
- **训练目标**：标准自回归语言建模损失 `L_AR`。

#### 阶段3：双负偏好优化（Dual-Negative Preference Optimization, DNPO）
- **构造偏好数据集RSICP**（20k图像对）：
  - **策略1（信息过滤选择）**：由RSICCLLM-sft生成K个候选描述，提取关键词（object、direction、change verb、quantity），计算与真实描述的平均BLEU-1/2和ROUGE-1相似度，选择中等相似度样本作为负样本（既不过难也不过易）。
  - **策略2（替换生成）**：随机替换真实描述中的关键词（从同类别池中选取），生成语义错误但语法自然的负样本。
- **优化目标**：`L = L_DPO + λ * L_S`，其中L_DPO为标准直接偏好优化，L_S为反向KL散度约束，防止策略模型偏离参考模型过远，稳定训练。
