## 摘要
测试时自适应（TTA）旨在仅使用无标签的流式数据将模型对齐到变化的测试域。大多数现有方法隐式推断单一的全局域分布，忽略了现实世界域偏移的多维性和样本特异性，导致自适应脆弱。我们提出DOME，一种有效的域编码器，以零样本方式显式建模每个样本的域。DOME利用视觉-语言预训练提取密集的连续表示，将域参数化为分布变量，并引入动量更新的稀疏域库以实现解耦监督。通过将这些显式域线索注入下游模型，即使是基本的熵最小化TTA策略也能在ImageNet-C、ImageNet-R和ImageNet-Sketch上达到最先进性能，超越复杂的TTA方法。我们的结果表明，鲁棒的自适应并非源于复杂的自适应算法，而是来自显式的、结构化的域表示。

## Abstract
Test-time adaptation (TTA) aims to align a model to shifting test domains using only unlabeled streaming data. Most existing methods implicitly infer a single global domain distribution, ignoring the multidimensional and sample-specific nature of real-world domain shifts, leading to fragile adaptation. We propose DOME, an effective domain encoder that explicitly models each sample's domain in a zero-shot manner. DOME leverages vision-language pretraining to extract dense, continuous representations, parameterizes domains as distributional variables, and introduces a momentum-updated sparse domain bank for disentangled supervision. By injecting these explicit domain cues into downstream models, even a basic entropy-minimization TTA strategy achieves state-of-the-art performance across ImageNet-C, ImageNet-R, and ImageNet-Sketch, outperforming complex TTA approaches. Our results demonstrate that robust adaptation stems not from intricate adaptation algorithms, but from explicit, structured domain representation.

- **问题**：测试时自适应（TTA）旨在利用无标签流式数据将模型适应到不断变化的测试域。现有方法大多隐式推断单一全局域分布，忽略了现实域偏移的多维性和样本特异性，导致自适应脆弱且易陷入局部最优。
- **背景**：视觉世界包含丰富多样的连续域（如气候、图像损坏、风格迁移），模型泛化困难。早期TTA方法通过更新归一化层统计量或自训练来隐式搜索域表示，但无法捕捉每个样本独有的域特征。

## 二、论文提出的方法论：核心思想、关键技术细节
### 核心思想
- 构建域编码器DOME，能零样本地为任意下游样本生成域表示。通过Sparse-to-Dense（S2D）学习框架训练，包括两个组件：稀疏标签域解耦（SDD）和密集域表示学习（DDRL）。在测试时，将DOME注入下游ViT模型，仅用简单熵最小化即可实现SOTA。

### 关键技术细节
1. **域编码器DOME**：
   - 基于冻结的CLIP图像和文本编码器初始化，融合图像特征和部分掩码的域文本提示（训练时随机遮蔽20%文本嵌入，推理时用零嵌入代替），经过6层Transformer得到融合图像表示。
   - 将域表示为预测的通道均值μ和标准差σ，由LayerNorm和线性层得到。

2. **稀疏标签域解耦（SDD）**：
   - 从CLIP图像编码器中间令牌去除位置编码，计算通道统计量并重参数化为潜变量z_img。
   - 通过字典解码器将z_img映射到高维过完备空间，得到稀疏表示d_k，用动量更新（m=0.9）维护域库原型p_k。
   - 应用L1稀疏正则化，并利用JumpReLU（阈值β可学习）对原型进行元素级稀疏化，得到干净的域表示z_dom。

3. **密集域表示学习（DDRL）**：
   - 将DOME输出的密集表示ẑ（从μ̂,σ̂重参数化）通过字典投影层得到ẑ_proj。
   - 使用三种损失对齐：
     - MSE损失：回归ẑ_proj到z_dom。
     - 辅助损失：对齐ẑ_proj到原始z_img。
     - 多正例InfoNCE损失：促进域内紧凑、域间分离。
   - 总损失：L_S2D = λ_sp L_sp + λ_MSE L_MSE + λ_NCE L_NCE + λ_aux L_aux，超参数设为1.0, 1.0, 0.1, 0.1。

4. **测试时自适应集成**：
   - 冻结DOME，将原始图像和弱增强图像分别输入DOME得到域统计量，通过轻量MLP适配器生成仿射参数(g,b)，注入ViT每个块的LayerNorm后（调制公式 x = x·(1+g)+b）。
   - 推理时仅更新适配器和LayerNorm仿射参数，ViT其他权重及DOME冻结。
   - TTA损失：条件熵L_ent + 边际熵L_marg（λ_ent=1.4, λ_marg=0.8）。

## 六、论文的主要结论与发现
1. DOME显式建模域表示，使简单熵最小化TTA达到SOTA（ImageNet-C 66.7%，ImageNet-R 68.7%，ImageNet-Sketch 53.9%）。
2. 域表示的质量比自适应算法复杂性更重要：DOME+基本TTA超越需要复杂正则化/记忆的方法。
3. 文本引导（CLIP文本提示）和稀疏激活（JumpReLU）对有效域解耦不可或缺。
4. DOME可即插即用提升现有TTA方法（如TENT+0.9%，SAR+1.8%，CoTTA+1.8%，DeYO+0.3%）。
5. 注意力图和t-SNE可视化表明DOME产生样本特定的域聚类，且能根据域复杂度自适应聚焦区域。
