## 摘要
连续时间动态图（CTDG）为捕获演化关系数据中的细粒度时间模式提供了更丰富的框架。长程信息传播是学习表征时的关键挑战，其中需要在长时间跨度内保留和更新信息。现有方法限制模型捕获单跳或局部时间邻域，无法捕获多跳或全局结构模式。为解决这一问题，我们从基本原理出发，为连续时间动态图推导了一个参数高效的状态空间建模框架（CTDG-SSM）。我们首先引入了连续时间拓扑感知高阶多项式投影算子（CTT-HiPPO），这是一种基于记忆的HiPPO新重构方法，用于联合编码时间动态和图结构。CTT-HiPPO的解通过将经典HiPPO解投影到拉普拉斯矩阵的多项式上获得，产生拓扑感知的记忆更新，这对应于CTDG的等效状态空间公式（CTDG-SSM）。然后，使用零阶保持方法获得计算高效的离散公式以进行模型实现。在动态链接预测、动态节点分类和序列分类的基准测试中，CTDG-SSM实现了最先进的性能。值得注意的是，在需要长程时空（LRT）推理的数据集上，它取得了较大的性能提升。

## Abstract
Continuous-time dynamic graphs (CTDGs) provide a richer framework to capture fine-grained temporal patterns in evolving relational data. Long-range information propagation is a key challenge while learning representations, wherein it is important to retain and update information over long temporal horizons. Existing approaches restrict models to capture one-hop or local temporal neighborhoods and fail to capture multi-hop or global structural patterns. To mitigate this, we derive a parameter-efficient state-space modeling framework for continuous-time dynamic graphs (CTDG-SSM) from first principles. We first introduce continuous-time Topology-Aware higher order polynomial projection operator (CTT-HiPPO), a novel memory-based reformulation of HiPPO to jointly encode temporal dynamics and graph structure. The solution from CTT-HiPPO is obtained by projecting the classical HiPPO solution through a polynomial of the Laplacian matrix, yielding topology-aware memory updates that admit an equivalent state-space formulation for CTDGs (CTDG-SSM). Then a computationally efficient discrete formulation is obtained using the zero-order hold approach for model implementation.   Across benchmarks on dynamic link prediction, dynamic node classification, and sequence classification, CTDG-SSM achieves state-of-the-art performance. Notably, it achieves large performance gains on datasets that require long range temporal (LRT) and spatial reasoning.

## 1. 核心问题与整体含义（研究动机与背景）

连续时间动态图（CTDG）以时间戳事件流的形式建模演化关系数据，比离散时间动态图保留了更细粒度的时序信息，在金融、医疗、电商和社交网络等领域有重要应用。然而，现有CTDG表征学习方法面临两大挑战：

- **长程时间依赖（LRT）**：需要在长时间跨度内保留和更新节点状态，防止信息衰减。
- **长程空间依赖（LRS）**：需要捕获多跳邻居的全局结构模式，而非仅局限于1‑跳局部邻域。

现有方法可分为事件驱动模型（如TGN、JODIE）和序列模型（如DyGFormer、DyGmamba）。前者主要捕捉短期局部模式，后者虽能处理LRT，但依靠1‑跳邻域补丁，无法有效捕获LRS。目前尚无方法能同时保证LRT和LRS，本文即针对这一空白提出统一框架。

## 2. 方法论：核心思想、关键技术细节与算法流程

### 2.1 核心思想
本文从第一性原理出发，将HiPPO（高阶多项式投影）扩展到图结构上，提出**连续时间拓扑感知高阶多项式投影算子（CTT‑HiPPO）**，再将CTT‑HiPPO的演化等价转化为状态空间模型（CTDG‑SSM），从而实现联合编码时间动态和图拓扑。

### 2.2 关键技术细节

- **CTT‑HiPPO**：将节点信号分解为图滤波器（拉普拉斯矩阵的多项式）与正交多项式基函数的线性组合，通过最小化残差得到系数矩阵 \(H_\tau = p(L_\tau)^{-1} H_{\tau}^{\text{HiPPO}}\)。即经典HiPPO系数经逆多项式图滤波得到拓扑感知的记忆表示。
- **CTDG‑SSM**：利用HiPPO系数满足的线性ODE，结合图滤波器随时间的变化，推导出连续时间状态空间方程（定理4.1）：
  \[
  \frac{dH_s}{ds} = -H_s A^\top M(s) - p(L_s)^{-1}\frac{dp(L_s)}{ds} H_s + p(L_s)^{-1}X(s) B^\top M(s)
  \]
  其中 \(A,B\) 由正交多项式决定，第二项刻画图结构变化对记忆的影响。
- **离散化**：采用零阶保持（ZOH）假设输入分段常数，得到离散递归（定理4.2）：
  \[
  H[k+1] = \bar{A}_{L[k]} H[k] \bar{A} + \bar{B}(L[k], X[k])
  \]
  其中 \(\bar{A}_{L[k]}\) 和 \(\bar{B}\) 依赖于当前及前一时刻的拉普拉斯矩阵。

### 2.3 算法流程（文字说明）

1. **子图采样**：将连续事件流按时间顺序分批，每批B个连续事件。对每个事件节点采样至多\(N_u\)个最近时间邻域，构造批级拉普拉斯矩阵 \(L_B[k]\) 和 \(L_B[k-1]\)。
2. **节点特征编码**：将节点原始特征、边特征、时间间隔编码等拼接，经2层神经网络映射到d维隐空间，得到 \(\tilde{X}[k]\)。
3. **CTDG‑SSM层**：对 \(\tilde{X}[k]\) 执行RMS归一化，利用离散更新公式计算新隐状态 \(H[k+1]\)，再通过残差连接和GeLU激活输出下一层输入。可堆叠多层。
4. **记忆更新**：仅更新当前批活跃节点的状态，其余节点保持原值。
5. **解码**：对于链接预测，将两个节点的隐状态与时间差编码拼接后经线性层输出分数；对于节点分类，直接对目标节点隐状态进行线性映射。

## 7. 优点

- **第一性原理建模**：从HiPPO出发，自然引入图拓扑，避免了启发式设计。
- **统一长程时空建模**：单个框架同时处理LRT和LRS，弥补了现有方法缺陷。
- **理论保障**：给出扰动鲁棒性和置换等变性的数学证明。
- **计算高效**：采用批级子图采样控制活跃节点数，离散更新复杂度低于Transformer/Mamba类方法。
- **轻量参数**：主要参数仅多项式系数和状态矩阵，总参数量远小于DyGFormer等。
- **离散化方案适配**：ZOH方法便于实现，且通过高斯‑勒让德求积近似积分，保持精度。

## 8. 不足与局限

- **图滤波器表达力受限**：当前仅使用固定阶多项式（一阶或二阶），未来可探索通过学习图神经网络自适应滤波器。
- **谱约束**：要求多项式在[0,2]上非零以保证\(p(L)\)可逆，可能限制滤波器设计。
- **缺失特征处理**：实验假设节点/边特征完全观测，未考虑缺失或噪声特征场景。
- **边删除场景**：虽然附录讨论了通过反向构造拉普拉斯差分来处理边删除，但实验仅在边添加型数据集上验证。
