# Learning Cross-View Semantic Priors for Single-Reference Unseen Object Pose Estimation 

Jiahong Chen, Jinghao Wang, Ziwen Wang, Zi Wang ∗, Banglei Guan, Member, IEEE , and Qifeng Yu 

Abstract —Single-reference unseen object 6D pose estimation reduces object onboarding by estimating poses of arbitrary novel objects from only one reference view. Recent correspondence-based pipelines have achieved robust performance with vision foundation model (VFM) features. However, they typically treat these features as intra-view descriptors, leaving dense visual-semantic cues, including appearance, structure, and context, insufficiently exchanged across views before geometric decod-ing. Consequently, the decoded point features may lack joint semantic and geometric discriminability, making correspondence estimation still difficult in challenging cases. Instead of processing features independently, we build the correspondence pipeline around an early cross-view semantic prior. Specifically, cross-view semantic interaction (CVSI) enables dense query and reference VFM tokens to exchange semantic context and form a cross-view prior. Nevertheless, direct CVSI may disturb the VFM token structure, while the resulting semantic prior still needs 3D representation consistency for rigid correspondence. To make this CVSI prior reliable for 3D correspondence learning, we in-troduce two complementary training-time constraints: the intra-view structure preservation (IVSP) loss preserves the original intra-view token affinity structure during interaction, while the reference-anchored geometric consistency (RAGC) loss enforces spatial representation consistency of decoded point features. The final pose is recovered from learned correspondences through weighted SVD. We further construct a challenging view-pair protocol from the BOP Challenge datasets YCB-V and TUD-L to evaluate robustness in difficult matching scenarios. Extensive experiments on six benchmarks under different view-pair settings show that our method achieves state-of-the-art performance while maintaining comparable inference speed. Code, data, and models will be available at chenjiahongbq.github.io/LCVSP .

Index Terms —Unseen object pose estimation, single-reference pose estimation, cross-view semantic priors, vision foundation models.

Category-level methods improve transfer by learning shared priors within predefined semantic classes [23–31], yet remain constrained by category-level supervision and cannot directly handle arbitrary objects beyond the training taxonomy [21] [21]. These limitations motivate unseen object pose estimation, which targets arbitrary novel objects beyond known instances and predefined categories [32–37]. To further reduce object onboarding, recent studies have moved toward the single-reference setting [38–48]. In this setting, only one RGB-D observation of the target object is available as the reference. A common solution is to follow a correspondence-based formulation [38–40], where the relative pose is estimated from correspondences between the query and reference observations. The central challenge is to make these correspondences reliable under large viewpoint changes and noisy observations. As illustrated in Fig. 1(a), dense VFM feature maps are extracted independently from the query and refer-ence images, projected onto sampled foreground point clouds, and then processed by a geometric decoder [51] for point matching and pose recovery. This design has been effective for geometry-aware matching, but it does not fully condition dense visual-semantic cues across views before geometric decoding. Appearance, part structure, and contextual relations encoded by VFM tokens are therefore weakened when they are used only as independent view-wise descriptors. As a result, the decoded point features may lack joint semantic and geometric discriminability: they can encode local 3D consistency, but remain weak at distinguishing truly corresponding object re-gions from geometrically plausible yet semantically unrelated regions. This limitation becomes evident in two typical cases. First, under noisy segmentation masks, as shown in Fig. 1(a), the sampled foreground point cloud may contain nearby objects whose local geometry resembles the target. If the decoder mainly relies on geometry at this stage, these distractor points can receive high matching salience and lead to false correspondences. Second, under large viewpoint changes, the visible overlap between the query and reference observations becomes sparse and spatially ambiguous, as shown in Fig. 2. In both cases, reliable matching requires a cross-view semantic prior before geometric decoding, so that point-level correspon-dence learning can be guided by reference-conditioned visual semantics rather than relying only on sparse geometric cues. Motivated by this observation, we build the correspon-dence pipeline around an early cross-view semantic prior. As shown in Fig. 1(b), our method introduces cross-view semantic interaction (CVSI) at the dense VFM token level before projection and geometric decoding. Instead of treating query and reference VFM features as independent intra-view descriptors, CVSI allows dense tokens from the two views to exchange semantic context and condition each other. The resulting tokens form a visual semantic prior conditioned cross views. When fused with geometry-aware embeddings in the decoder, this prior encourages the decoded point features to be both semantically selective and geometrically consistent for correspondence estimation. However, learning such a prior is non-trivial because the query and reference observations are unposed, partially overlapping, and may contain mask-induced distractors. Direct dense token interaction can therefore cause excessive feature mixing and over-smooth the spatial organiza-tion encoded by the VFM. In addition, the learned prior should be translated into spatially consistent point features to benefit rigid 3D correspondence estimation. To address these issues, we introduce two complementary training-time constraints that jointly make the CVSI prior reliable for 3D correspon-dence learning. The intra-view structure preservation (IVSP) loss preserves the original intra-view token affinity structure during cross-view interaction, avoiding over-smoothed spatial organization. The reference-anchored geometric consistency (RAGC) loss enforces spatial representation consistency of decoded point features in a shared reference frame, mak-ing the semantic prior compatible with rigid correspondence estimation. The final pose is recovered from learned corre-spondences through weighted SVD. Together, CVSI, IVSP, and RAGC are designed to improve correspondence reliability under large viewpoint changes and noisy masks. To evaluate this capability, we further construct a challenging view pair protocol from the BOP Challenge datasets YCB-V [8] and TUD-L [52]. Extensive experiments on LM-O [53], TUD-L [52], YCB-V [8], REAL275 [23], Toyota-Light [52], and LINEMOD [54] demonstrate that our method achieves state-of-the-art performance across different view pair settings while maintaining comparable inference speed. A. Instance-level and Category-level Object Pose Estimation 

Instance-level 6D object pose estimation has been exten-sively studied for known rigid objects, typically under the as-sumption that CAD models and object-specific annotations are available. Early methods relied on template matching [54] or local geometric descriptors such as FPFH [55] and PPF [56]. With the development of deep learning, later methods learn pose representations through direct pose prediction [8], key-point voting [10, 13], dense correspondence prediction [9, 11, 12, 14, 19], RGB-D feature fusion [57–59], or iterative refinement [60, 61]. Despite strong performance, instance-level methods remain tied to known objects or object models [21, 22]. Category-level pose estimation relaxes this dependence by estimating the pose and size of unseen instances within predefined categories [21]. A common strategy is to learn a category shared representation, such as normalized object coordinates [23], deformable shape priors [24], or canonical shape spaces [25]. Subsequent works improve shape adapta-tion [26, 28], pose consistency [27], equivariant representation learning [29], and semantic geometry fusion [30, 62]. Recent studies further address intra-class variation [63], global con-text [64], causal robustness [65], and outlier-aware correspon-dence fitting [66]. Nevertheless, category-level methods still rely on category supervision and predefined semantic classes. 

B. Novel Object Pose Estimation 

Novel object pose estimation aims to recover the 6D pose of arbitrary objects beyond known instances and predefined categories. One line of work addresses this setting by using object models, rendered templates, or generalizable matching at test time. MegaPose [33] follows a render-and-compare paradigm, where CAD-based renderings are retrieved and refined for pose estimation. Templates Revisited [67] revisits template-based pose estimation and studies its generalization to new objects and robustness to occlusion. GigaPose [68] improves CAD-based novel object pose estimation with dis-criminative rendered templates and a compact correspondence formulation. Recent methods further exploit vision foundation models [49, 50] and general segmentation priors [69]. SAM-6D [36] combines SAM-based object proposal generation with semantic, appearance, and geometric matching, and formulates pose estimation as partial-to-partial point matching. Found-Pose [34] leverages foundation features to establish corre-spondences for unseen objects, while FreeZe [70] explores a training-free zero-shot pipeline with geometric and vision foundation models. When CAD models are unavailable, another direction rep-resents the target object using multiple reference observa-tions [35, 71–74]. OnePose [71] reconstructs a sparse object model from a video scan using SfM, matches 2D query points to 3D SfM points, and estimates the object pose with PnP. OnePose++ [72] further removes the dependence on repeat-able keypoint detection by using keypoint-free matching and reconstructing a semi-dense object point cloud. Gen6D [73] assumes several posed reference images and estimates pose through detection, viewpoint selection, and refinement. Foun-dationPose [35] unifies model-based and model-free pose estimation by using either a CAD model or a small set of reference images. These methods improve generalization to novel objects, but they still typically rely on either CAD models or multiple reference views. Recent works therefore move toward the more constrained single-reference setting [38–48]. NOPE [41] predicts pose-conditioned discriminative embeddings from a single reference image and estimates the query pose by matching them to generated viewpoint embeddings. Oryon [43] and Horyon [44] study open-vocabulary relative pose estimation, where a text prompt identifies the target object across two scenes and visual-language features are used for cross-scene matching and 3D registration. One2Any [45] and CoordAR [47] pre-dict reference object coordinates from the query observation, converting single-reference pose estimation into alignment in a shared reference coordinate space. UNOPose [38] casts single-reference pose estimation as point cloud registration by combining VFM semantic descriptors with an SE(3)-invariant geometric representation for matching. COG [39] extends this correspondence-based formulation with confidence-aware optimal transport to obtain soft correspondences guided by point confidence. Unlike UNOPose [38] and COG [39], which mainly improve geometric decoding or correspondence assign-ment, we focus on feature construction before matching. Our method performs cross-view semantic interaction at the dense VFM token level, forming a cross-view semantic prior for geometry-aware correspondence learning. 

C. Point Cloud Registration 

Point cloud registration is closely related to single-reference unseen object pose estimation, as both require reliable corre-spondences between partial 3D observations. Classical reg-istration methods mainly rely on geometric matching. ICP and its variants [75–77] iteratively refine closest point asso-ciations, FPFH [55] describes local surface neighborhoods, and PPF [56] encodes oriented point pair relations for pose hypothesis generation. These methods provide useful geo-metric baselines, but are sensitive to outliers, repeated local structures, and limited overlap. Learning methods improve registration by estimating more discriminative correspondence cues. DCP [78] uses attention for feature matching and dif-ferentiable transformation estimation. Predator [79] predicts overlap-aware features to focus matching on shared regions under low overlap. GeoTransformer [51] encodes pairwise distances and angular relations to obtain geometric features that are invariant to rigid transformations for robust superpoint matching. Unlike generic registration, single-reference unseen object pose estimation operates on object centered partial RGB-D point clouds cropped by predicted masks. Noisy masks may introduce structured distractors from nearby objects, while large viewpoint changes make valid overlap sparse and am-biguous. In such cases, geometry alone is often insufficient. Appearance, local layout, and object context encoded by VFMs [49, 50] provide complementary cues for identifying reliable correspondences. This motivates introducing a cross-view semantic prior before geometry-aware matching. 

D. Cross-View Correspondence Learning 

Cross-view correspondence learning has been studied in image matching, object association, and visual geometry rea-soning. LoFTR [80] shows that transformer feature interaction can establish dense local correspondences without explicit key-point detection. O-MaMa [81] formulates ego-exo object cor-respondence as mask matching, using DINOv2 features [49] and cross-attention for object level alignment. V 2-SAM [82] adapts SAM2 to cross-view object correspondence by com-bining geometry-aware anchor prompts from DINOv3 [50] with visual prompts for appearance alignment. VGGT [83] further demonstrates that cross-view spatial consistency can be encoded through unified visual geometry representations. These studies suggest that explicit cross-view interaction is useful for transferring appearance, structure, and spatial cues across large viewpoint changes. However, most of them focus on image level correspondence, mask association, or general visual geometry reasoning. We explore such cross-view cues in single-reference unseen object pose estimation, where they must be further grounded by geometric consistency for RGB-D correspondence learning and rigid pose recovery. A. Problem Formulation 

Given a query RGB-D image [Iq | Dq ] and a single reference RGB-D image [Ir | Dr ] of the same unseen rigid object, our goal is to estimate the relative object pose from the query view to the reference view, denoted by ∆Tq→r ∈ SE (3) .The reference image is accompanied by an object mask M r

indicating the target object. For the query image, the object mask M q can be obtained either by a segmentation foundation model [69] in the full pipeline or by ground-truth annotation when evaluating the pose estimator in isolation. Using M q and M r , we filter out background pixels, back-project the masked depth maps into 3D space, and sample the same number of foreground points to construct a query point cloud Pq = {pqi }Ni=1 and a reference point cloud Pr =

{pri }Ni=1 , where pqi , p ri ∈ R3. Our method takes these sampled partial observations as input and predicts their relative rigid transformation ∆Tq→r. If the absolute pose of the reference object is available, the absolute pose of the query object can be recovered by T q = ∆ Tq→r T r , where T r denotes the absolute pose of the reference object.

A. Experimental Setup 

Implementation Details. Our method is implemented in PyTorch [88]. For image encoding, we use a DI-NOv3 [50] pretrained ViT-Base backbone [89]. For geometric correspondence extraction, we follow prior single-reference pipelines [38, 39] and adopt a coarse-to-fine design based on GeoTransformer [51]. Following [38, 39], the network is trained on the standard MegaPose synthetic dataset [33] for the BOP unseen object pose estimation track [90]. This dataset contains about 2M images of about 50K objects from GSO [91] and ShapeNet [92]. We train the network for 440K steps with a batch size of 8 on the RTX 4090 GPU. Optimiza-tion is performed with Adam [93] and cosine annealing [94], using a base learning rate of 10 −4. Unless otherwise specified, the number of CVSI blocks is set to L = 3 , the number of attention heads is set to H = 8 , and the loss weights are fixed to λRAGC = 1 and λIVSP = 1 . In the coarse stage, we sample Nc = 196 points and generate 300 pose hypotheses. In the fine stage, we increase the number of sampled points to Nf = 2048 .

Datasets. We conduct experiments on six benchmark datasets: LM-O [53], TUD-L [52], YCB-V [8], Real275 [23], Toyota-Light [52], and LINEMOD [54]. Among them, LM-O, TUD-L, and YCB-V are used to evaluate pose estimation under the predefined reference-query view-pair protocol fol-lowing [38]. For these three datasets, object masks are obtained by SAM [69], and thus contain realistic segmentation noise. LM-O consists of 200 test images from a cluttered scene, where eight objects with occlusion are provided for evaluation. TUD-L contains three moving objects with mild occlusion and diverse lighting conditions. YCB-V is more challenging due to severe occlusion and sensor noise, and contains 900 test images from 12 scenes involving 21 objects. We further evaluate on Real275 and Toyota-Light under the protocol of Oryon [43], where each test set provides 2,000 reference-query pairs. For these two datasets, ground-truth object masks are used. Real275 contains 18 objects from six categories in di-verse indoor environments, such as tabletops and floor scenes, with substantial viewpoint variation. Toyota-Light contains 21 objects captured under challenging illumination conditions across different scenes. To further assess generalization to the reference view, we additionally evaluate single-reference pose estimation on YCB-V by using only the first frame as the reference, following [35, 45]. We also report results on LINEMOD [54] under the same single-reference protocol to evaluate robustness under large viewpoint changes. LINEMOD consists of 13 sequences, each containing about 1.2K images with ground-truth poses for a single object under clutter and mild occlusion. 

Challenging Benchmark. We further construct a challeng-ing view-pair benchmark based on YCB-V [8] and TUD-L [52]. For each query object in the test split, we randomly select a reference view from the training or validation split and constrain the relative rotation angle to the range of 60 to 90 degrees. For the object masks, we start from the segmentation results of UNOPose [38] and further inject noise by applying random morphological dilation. This benchmark is designed to evaluate robustness under large viewpoint changes and noisy masks. 

Evaluation Metrics. For LM-O, TUD-L, and YCB-V under the predefined reference-query view-pair protocol, we follow the BOP evaluation protocol and report average recall (AR) over the three standard BOP pose metrics, namely visible surface discrepancy (VSD), maximum symmetry-aware sur-face distance (MSSD), and maximum symmetry-aware pro-jection distance (MSPD) [52]. For Real275, Toyota-Light, and LINEMOD, we report ADD(-S) at 0.1d, where the ADD metric measures whether the average deviation of the trans-formed model points is below 10% of the object diameter 

d. For symmetric objects, ADD-S is used instead, where the error is defined as the average distance to the closest model point [54, 95]. For YCB-V under the first-frame reference setting, we report AUC of ADD and AUC of ADD-S following PoseCNN [8].

Recent single-reference methods benefit from RGB-D observations and query-reference reasoning. Coordinate prediction meth-ods such as CoordAR [47] estimate reference object coor-dinates and recover pose by alignment, but their accuracy can be affected when noisy masks introduce falsely sam-pled regions. Correspondence-based methods, including SAM-6D [36], SinRef-6D [40], UNOPose [38], and COG [39], recover pose through query-reference matching. Among them, COG improves the assignment stage by formulating corre-spondence estimation as confidence-aware optimal transport. In contrast, our method improves the representation before matching: CVSI introduces cross-view semantic interaction before geometric decoding, so that the decoded point features are conditioned by semantic compatibility across views before correspondence assignment. Thus, COG mainly improves how correspondences are assigned, whereas our method improves the features on which correspondences are computed. Quantitatively, our method achieves the best performance on all three datasets, with a mean AR BOP of 76.6%, compared with 70.9% for UNOPose [38] and 73.8% for COG [39]. The gain over COG is moderate on LM-O, where severe occlusion remains a limiting factor for all matching-based methods, but becomes more evident on TUD-L and YCB-V. In particular, our method reaches 86.7% AR BOP on YCB-V, outperforming UNOPose by 3.6% and COG by 6.2%. Since YCB-V contains stronger ambiguity from clutter, occlusion, sensor noise, and multiple objects, this result suggests that early cross-view semantic interaction improves the discriminability of corre-spondence features in challenging matching scenarios.

The improvement is also consistent on MSSD and MSPD, which increase from 88.4% to 90.1% and from 81.7% to 84.0%, respectively. This result shows that performing cross-view interaction before geometric decoding provides more discriminative features for correspondence estimation. In particular, the larger gains on MSSD and MSPD suggest that CVSI improves both 3D surface alignment and projec-tion accuracy, which are directly affected by correspondence quality. The RAGC-only variant improves AR BOP from 84.2% to 84.7% (B2 vs. B0). This moderate gain indicates that reference-anchored geometric supervision provides a useful geometric regularization signal, but it is not the main source of the overall improvement. This comparison is important because it controls for the effect of the auxiliary coordinate prediction head: adding the RAGC head and loss alone cannot SUBMITTED TO IEEE TRANSACTIONS ON PATTERN ANALYSIS AND MACHINE INTELLIGENCE 13 

explain the full gain of the proposed method. Instead, RAGC is most useful when it works with cross-view semantic condi-tioning, where it encourages the decoded point features to be consistent in a shared 3D reference frame. The role of IVSP is also clarified by comparing B1, B3, and B5. When CVSI and RAGC are used without IVSP, the AR BOP remains 85.7% (B5), the same as CVSI alone (B1). After adding IVSP, the performance increases to 86.2% (B3), with consistent gains on VSD, MSSD, and MSPD. This shows that IVSP is not merely an additional auxiliary loss, but a stabilizing regularizer for cross-view token interaction. By preserving the intra-view token affinity structure, IVSP prevents the interacted tokens from losing the original VFM organization, allowing the semantic prior introduced by CVSI and the geometric constraint imposed by RAGC to better complement each other. Finally, sharing the cross-attention weights between the query and reference branches further improves AR BOP from 86.2% to 86.7% (B4 vs. B3). This shared design encourages the two branches to follow a consistent interaction pattern and reduces redundant parameters. Compared with the DI-NOv3 baseline, the full model improves AR BOP by 2.5%, from 84.2% to 86.7%. The improvements are also observed across all three BOP metrics, with gains of 1.5, 2.7, and 3.4% on VSD, MSSD, and MSPD, respectively. These results support the complementary design of the proposed framework: CVSI provides the main cross-view semantic prior, RAGC grounds the decoded features in 3D reference coordinates, and IVSP stabilizes the token structure during interaction. Since RAGC and IVSP are used only during training, they improve feature learning without adding inference time computation. We therefore adopt the shared CVSI design with both RAGC and IVSP as the default configuration.

E. Failure cases 

Although our method improves correspondence reliabil-ity under large viewpoint changes and mask noise, it is still subject to the observability limits of single-reference correspondence-based pose estimation. As shown in Fig. 12, the first failure mode occurs when the query and reference views have near-zero visible overlap. In this case, very few regions can support valid cross-view correspondences, making it difficult for both semantic interaction and geometric match-ing to infer a stable rigid transformation. The second failure mode arises from highly incomplete object masks. When discriminative object parts are missing from the segmented observation, the input no longer provides sufficient object evidence for either dense token-level interaction or point-level correspondence learning. As a result, the learned semantic prior and the subsequent geometric decoder may both be guided by incomplete or biased object structures. These cases should therefore be interpreted as boundary conditions of the single-reference correspondence formulation. V. CONCLUSIONS This paper studied single-reference unseen object 6D pose estimation and identified late sparse cross-view interaction as a key limitation of existing correspondence-based pipelines. In these methods, dense VFM features are usually projected to sparse point features before cross-view reasoning, which limits the use of visual-semantic cues for correspondence learning. To address this issue, we proposed a geometry-aware cross-view semantic prior learning framework. CVSI introduces cross-view semantic interaction at the dense VFM token level before geometric decoding, ISP preserves the intra-view token structure during this interaction, and RAGC grounds the de-coded point features in a shared reference-anchored 3D space. Together, these components improve the joint semantic and geometric discriminability of correspondence features, leading to more reliable point cloud matching and weighted SVD pose estimation. Extensive experiments on six benchmark datasets under different view-pair settings, together with evaluations on a challenging view-pair protocol, demonstrate that the proposed method achieves state-of-the-art performance with comparable inference speed. 

Limitations and Future Work. Despite these improve-ments, the proposed method remains limited by the input evi-dence available in the single-reference setting. When the query and reference views have near-zero overlap, or when segmen-tation masks remove discriminative object regions, reliable correspondences become difficult to establish. These cases reflect the inherent observability limits of single-reference correspondence-based pose estimation. In future work, we plan to extend the framework to incorporate multiple query observations and explicitly model their geometric relation-ships. Such a direction may increase object evidence, alleviate failures caused by large viewpoint gaps and incomplete masks, and support joint reasoning over object geometry and multi-view poses.
