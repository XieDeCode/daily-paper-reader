## 摘要
旋转位置嵌入（RoPE）使注意力分数具有位置相对性，但值路径仍然对位置不敏感：无论值令牌与查询之间的距离如何，其发送的消息都是相同的。我们提出RoVE，一种无需参数修改的方法，通过将值与键同时旋转来使值具有位置敏感性，并证明它将RoPE注意力转化为注意力卷积。这一新视角统一了计算机视觉、机器人学和现代大语言模型架构中同一操作的多种独立表述。在124M和354M参数的GPT-2模型上进行的训练显示，相对于RoPE，RoVE在少样本上下文学习、分布外困惑度和长上下文检索方面一致取得实证增益，在需要长距离聚合的任务上改进最为明显。

## Abstract
Rotary Position Embeddings (RoPE) make attention scores position-relative but leave the value pathway position-blind: the message sent by a value token is the same regardless of its distance from the query. We propose RoVE, a parameter-free modification that makes values position-sensitive by rotating them simultaneously with keys, and show that it turns RoPE attention into attentive convolution. This new perspective unifies several independent formulations of the same operation across computer vision, robotics, and modern LLM architectures. Trained 124M and 354M GPT-2 models show consistent empirical gains over RoPE on few-shot in-context learning, out-of-distribution perplexity, and long-context retrieval, with the clearest improvements on tasks that require long-range aggregation.

### 1. 论文的核心问题与整体含义（研究动机和背景）

标准旋转位置嵌入（RoPE）使**查询-键（QK）** 电路的注意力分数依赖于相对位置，但**输出-值（OV）** 电路中的值变换仍然是位置无关的：无论值令牌与查询之间的距离如何，值投影矩阵 \(W_V\) 完全相同，导致模型无法根据相对距离调节要聚合的特征。这限制了长距离依赖建模能力。本文提出RoVE（Rotary Value Embeddings），通过将值向量也像键和查询一样进行旋转，**使值路径获得与QK电路相同的相对位置敏感性**，从而将RoPE注意力转化为**注意力卷积**（attentive convolution）。该工作统一了计算机视觉、机器人学和大语言模型中多个独立的类似构造，并首次在标准（非共享KV）语言模型中系统评估。

### 2. 论文提出的方法论：核心思想、关键技术细节、公式或算法流程（文字说明）

- **核心思想**：在标准RoPE注意力中，输出计算为：
  \[
  y_i = \sum_{j \in \mathcal{N}(i)} A_{ij} W_V x_j
  \]
  其中 \(A_{ij}\) 是依赖于相对偏移 \(\delta = j-i\) 的注意力权重，但值映射 \(W_V\) 与偏移无关。RoVE将值路径修改为：
  \[
  \tilde{y}_i = R^{-1}_i \sum_{j \in \mathcal{N}(i)} A_{ij} R_j W_V x_j = \sum_{j \in \mathcal{N}(i)} A_{ij} \; \underbrace{R_{\delta} W_V}_{\psi_\delta} \; x_j
  \]
  其中 \(R_t\) 是RoPE中的块对角旋转矩阵。这样，每个偏移 \(\delta\) 对应的值核变为 \(\psi_\delta = R_\delta W_V\)，即**一个共享的 \(W_V\) 通过旋转矩阵族获得偏移依赖**。

- **注意力卷积视角**：上式正是**注意力卷积**的定义：用内容依赖的标量 \(A_{ij}\) 乘以偏移依赖的核矩阵 \(\psi_\delta\)。标准RoPE是特例（\(\psi_\delta \equiv W_V\)）。

- **矩阵混合器视角**：RoVE将混合器的 \(d\times d\) 块结构从因子化的 \(A_{ij} W_V\)（Kronecker结构）变为块Toeplitz结构（同一对角线共享相同的旋转后核），与卷积一致。

- **局部帧变换视角**：先通过 \(R_j\) 将每个值变换到全局坐标系，聚合后再通过 \(R^{-1}_i\) 转回查询的局部坐标系。

- **效率**：旋转操作复杂度 \(O(nd)\)（独立作用于每个令牌的 \(d/2\) 个通道对），相对于 \(O(n^2 d)\) 注意力计算可忽略，且不引入额外参数，兼容FlashAttention。

### 6. 论文的主要结论与发现

- RoVE在训练分布内（ICL准确率、困惑度）**一致优于RoPE**，证明值路径旋转提供有用的归纳偏置。
