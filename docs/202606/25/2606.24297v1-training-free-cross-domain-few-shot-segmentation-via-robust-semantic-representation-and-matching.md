## 摘要
跨域少样本分割（CD-FSS）旨在将从源域学到的知识迁移到不同的目标域，仅用少量标注样本对未见目标类别进行分割。尽管现有方法取得了显著进展，但它们仍然依赖于训练或微调过程，这会产生高昂的计算成本并有过度拟合的风险。我们观察到，当将强大且通用的视觉基础模型整合到这些方法中时，其性能仅有轻微提升，甚至由于过拟合而下降。为了解决这个问题，我们消除了可训练参数，并提出了一种无训练框架，以避免训练开销和过拟合。基于自监督视觉编码器DINOv3，我们的框架通过三个核心模块应对跨域挑战。首先，语义感知特征再融合（SAFR）模块识别并重新融合强调语义模式的特征，生成具有增强语义可区分性的表示。此外，自适应支持增强（ASE）模块通过稳健的查询信息聚合缩小支持和查询之间的语义差距。最后，混合原型匹配（HPM）模块整合来自不同原型的匹配结果，以适应跨域不同的语义复杂性。在四个目标域数据集上的大量实验表明，我们的方法在无需任何训练的情况下实现了CD-FSS的最新性能。

## Abstract
Cross-domain Few-shot Segmentation (CD-FSS) aims to transfer knowledge learned from source domain to distinct target domains, segmenting unseen target classes with only a few annotated samples. Although existing methods have made significant progress, they still rely on training or fine-tuning processes, which incur high computational costs and risk overfitting. We observe that when powerful and general-purpose vision foundation models are incorporated into these methods, their performance shows only marginal improvement or even degrades due to overfitting. To address this, we eliminate trainable parameters and propose a training-free framework to avoid both training overhead and overfitting. Built upon the self-supervised vision encoder DINOv3, our framework addresses cross-domain challenges through three core modules. First, the Semantic-aware Feature Re-fusion (SAFR) module identifies and re-fuses features that emphasize semantic patterns, generating representations with enhanced semantic discriminability. Additionally, the Adaptive Support Enhancement (ASE) module narrows semantic gaps between support and query through robust query information aggregation. Finally, the Hybrid Prototype Matching (HPM) module integrates matching results from diverse prototypes to adapt to varying semantic complexity across domains. Extensive experiments on four target domain datasets demonstrate that our method achieves state-of-the-art performance in CD-FSS without any training.

- **现有问题**：当前 CD-FSS 方法（原型法、亲和法、源域微调法）均依赖源域训练或目标域微调，导致：
  - **高计算开销**：需要大量 GPU 训练时间。
  - **过拟合风险**：尤其在引入强大视觉基础模型（VFM，如 DINOv3）后，可训练参数容易在训练数据上过拟合，性能提升有限甚至下降（如图 1(b) 所示）。
- **核心动机**：作者发现基于 VFM（DINOv3）的简单基线在跨域场景中已展现巨大潜力，说明 VFM 本身具备强大跨域特征表达能力。因此，提出完全 **无训练** 的框架，避免训练开销和过拟合。

## 2. 方法论

### 核心思想
利用 DINOv3 作为固定编码器，通过三个无训练模块增强语义表示的判别性并实现鲁棒匹配，生成精确的查询预测。

### 关键技术细节

#### （1）语义感知特征再融合（SAFR）
- **动机**：DINOv3 最后一层特征包含大量位置敏感通道（高方差），弱化了语义判别性。分析表明，某些中间层注意力输出（Flattn）更关注语义模式，但经残差连接会传播位置噪声。
- **实现**：
  - 提取所有层的 `Flattn` 和最终输出 `Flast`。
  - 计算每层位置敏感通道的方差比例 `rlattn` 和 Fisher 判别比 `flattn`。
  - 识别“位置感知层集” `Lpos`（方差比例高于平均值），然后在相邻位置感知层之间选取方差比例最小的层作为“代表性语义感知层” `Lsem`。
  - 加权融合 `Lsem` 中的特征，权重 `ωl = (flattn / rlattn) / sum(..)`，同时最小化位置噪声并最大化语义判别性。
  - 支持图像与查询图像共享从支持特征计算的层选择和权重。

#### （2）自适应支持增强（ASE）
- **动机**：支持和查询之间的实例差异导致特征语义不一致，需要在不引入可学习参数的情况下缩窄差距。
- **实现**：
  - 从支持原型与支持特征计算相似度图，利用支持 GT 掩码选择最优阈值 `t*`，再通过稳健化（取大于 0.75×t* 的阈值均值）得到 `trob`。
  - 用 `trob` 二值化查询相似度图，得到稳健查询前景/背景掩码。
  - 使用**带掩码的交叉注意力**（无需投影层）：仅允许支持前景与查询前景（支持背景与查询背景）交互，防止特征混淆。
  - 聚合后的特征与原始支持特征平均得到增强支持特征。

#### （3）混合原型匹配（HPM）
- **动机**：不同目标域语义复杂度各异，单一原型类型（全局、区域或像素级）难以通用。
- **实现**：
  - 从增强支持特征中提取三种原型集：
    - 全局原型：通过掩码平均池化得到。
    - 区域原型：对支持前景特征使用 k-means++ 聚类（`Nr=9`），背景聚类 `2Nr` 个。
    - 像素级原型：每个支持前景/背景像素作为单独原型。
  - 每种原型分别与查询特征计算余弦相似度，取逐像素最大值得到 `M*`。
  - 根据每种原型的伪查询掩码计算平均前景相似度 `μm,f` 与平均背景相似度 `μm,b`，差值 `ωm` 作为置信度和融合权重，加权融合后比较得到最终预测。

1. **无训练框架达到 SOTA**：在四个目标域上，1-shot 平均 mIoU 68.39%，5-shot 73.08%，超越所有现有训练型和无训练方法。
