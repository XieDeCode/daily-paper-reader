## 摘要
跨域少样本分割（CD-FSS）旨在从源域中丰富的标注样本中学习可泛化的分割能力，从而在目标域中仅使用少量标注样本即可准确分割新类别。现有的CD-FSS方法主要关注缓解由风格差异引起的特征分布偏移，而忽略了跨域类别语义粒度与判别属性的显著差异，导致支持-查询匹配中出现两种关键退化：语义过度对齐和属性过度对齐。为此，我们提出了双重层级聚合网络（DHANet），包含三个关键模块。首先，层级空间聚合（HSA）模块沿空间维度对像素特征进行多尺度区域聚合，生成层级语义增强特征以缓解语义过度对齐。其次，层级通道聚合（HCA）模块沿通道维度进行多尺度属性聚合，生成层级属性增强特征以缓解属性过度对齐。最后，我们提出在线概率语义库（OPSB），在推理过程中从查询预测逐步构建并更新类别概率分布，并采样多个伪原型作为额外支持信息以缓解支持样本不足。在四个目标域数据集上的大量实验表明，我们的方法达到了最先进的性能。

## Abstract
Cross-domain Few-shot Segmentation (CD-FSS) aims to learn generalizable segmentation capability from abundant annotated samples in the source domain, enabling accurate segmentation of novel classes in the target domain with only a few annotated samples. Existing CD-FSS methods mainly focus on mitigating feature distribution shifts caused by style gaps while ignoring significant differences in class semantic granularity and discriminative attributes across domains, leading to two key degradations in support-query matching: semantic over-alignment and attribute over-alignment. To this end, we propose the Dual Hierarchical Aggregation Network (DHANet), which comprises three key modules. First, the Hierarchical Spatial Aggregation (HSA) module performs multi-scale region aggregation of pixel features along the spatial dimension, generating hierarchical semantic-enhanced features to alleviate semantic over-alignment. Additionally, the HCA module conducts multi-scale attribute aggregation along the channel dimension, generating hierarchical attribute-enhanced features to mitigate attribute over-alignment. Finally, we propose the Online Probabilistic Semantic Bank (OPSB), which progressively constructs and updates class probability distributions from query predictions during inference, and samples multiple pseudo-prototypes as additional support information to mitigate insufficient support. Extensive experiments on four target-domain datasets demonstrate that our method achieves state-of-the-art performance.

# 跨域少样本分割的层级空间与通道聚合：详细中文总结

## 1. 核心问题与研究动机
- **背景**：跨域少样本分割（CD-FSS）旨在利用源域（如PASCAL VOC）中大量标注数据训练模型，使其能泛化到目标域（如卫星图像、医学图像等）的新类别，且目标域只有少量标注样本。
- **核心问题**：现有CD-FSS方法主要关注缓解因风格差异导致的特征分布偏移，却忽略了跨域类别在**语义粒度**（如源域只区分“猫”整体，目标域需区分“猫头”和“猫身”）和**判别属性**（如源域不敏感的颜色属性，在目标域皮肤镜图像中却是关键判别线索）上的显著差异。这导致了两种关键退化：
    - **语义过对齐**：源域训练迫使同一类别像素特征对齐到一个原型，丧失了多粒度语义判别能力。
    - **属性过对齐**：跨图像通道对齐促使源域不敏感属性对应的通道退化，丢失了对目标域重要属性的敏感性。
- **本文动机**：首次同时关注语义过对齐和属性过对齐，通过双层次聚合网络（DHANet）缓解这两种退化。

## 2. 方法论
### 2.1 核心思想
- 分别沿**空间维度**和**通道维度**进行层次化聚合，捕获不同粒度下的语义和属性表征，将训练时的硬对齐（单一粒度）转化为软对齐（多粒度），同时为测试提供多粒度信息以适应不同域的需求。

### 2.2 关键技术细节

#### (1) 层级空间聚合（HSA）
- 设计为多阶段（\(L_s\) 阶段）架构，每个阶段使用一组可学习的**空间槽**（Spatial Slots）进行特征聚合。
- 输入端：高层深度特征 \(\mathbf{F}_h \in \mathbb{R}^{H \times W \times C}\)。
- 流程：
    - 交叉注意力：以空间槽为查询，\(\mathbf{F}_h\) 为键值，聚合初始信息。
    - 槽注意力：迭代更新得到区域原型 \(\hat{\mathbf{S}}_{sp}^i \in \mathbb{R}^{N_{sp}^i \times C}\) 和注意力图 \(\hat{\mathbf{A}}_{sp}^i\)。
    - 通过反向掩码平均池化（RMAP）将每个区域原型应用到其对应像素，得到区域特征图 \(\mathbf{F}_i\)。
    - 所有阶段区域特征图与原始特征相加，得到层次语义增强特征 \(\tilde{\mathbf{F}}\)。
- 目的：使支持-查询特征能在多个语义粒度上对齐，缓解语义过对齐。

#### (2) 层级通道聚合（HCA）
- 类似HSA的层次化结构（\(L_c\) 阶段），但沿通道维度进行聚合。
- 输入：将 \(\tilde{\mathbf{F}}\) 与浅层特征 \(\mathbf{F}_l\) 相加得到 \(\hat{\mathbf{F}}\)，并转置为 \(\mathbb{R}^{C \times HW}\)。
- 每个阶段使用**通道槽**（Channel Slots）聚合通道信息，得到属性原型 \(\hat{\mathbf{S}}_{ch}^i \in \mathbb{R}^{N_{ch}^i \times HW}\)。
- 与HSA不同：不使用RMAP填充，而是将所有阶段的属性原型与原始特征沿通道维度拼接，得到层次属性增强特征 \(\bar{\mathbf{F}} \in \mathbb{R}^{H \times W \times C'}\)。
- 目的：保留通道多样性，缓解通道硬对齐导致的属性退化。

#### (3) 在线概率语义库（OPSB）
- 在测试时，从每一轮的查询预测中提取高置信度特征，构建并更新每个类别的**高斯分布**（均值和方差）。
- 从最新分布中采样 \(P\) 个伪原型，与查询特征计算余弦相似度，得到最大和平均前景置信图，与主分支结果融合。
- 目的：在不进行在线微调（避免知识遗忘）的情况下，为缺乏支持信息提供额外、多样化的类别先验。

#### (4) 损失函数
- 主分支：二元交叉熵损失 \(L_{\text{main}}\)。
- 辅助：自分割损失 \(L_{\text{ssp}}\)（来自SSP）。
- 槽正交约束损失 \(L_{\text{orth}}\)（防止不同槽退化到相似表示）。
- 总损失：\(L = L_{\text{main}} + L_{\text{ssp}} + \lambda L_{\text{orth}}\)，\(\lambda = 0.1\)。

### 2.3 训练与推理流程
- 训练时：使用全局风格随机化（GSR）对输入图像进行扰动，然后经共享编码器提取特征，依次通过HSA和HCA，再通过密集支持-查询匹配得到主分支预测，计算损失。
- 测试时：额外启用OPSB辅助分支，融合主、辅分支置信图，自适应阈值得到最终分割掩码。

## 6. 主要结论与发现
- DHANet在**1-shot**和**5-shot**设置下均超过所有对比方法，平均mIoU分别达到70.93%和74.26%，比之前SOTA（HSL）分别高出2.69%和1.94%。

- HSA和HCA互补，OPSB能进一步提升性能同时避免在线微调的知识遗忘风险。
- 通道相关性分析表明HCA能有效降低通道冗余，保留属性多样性。

## 7. 优点
1. **问题洞察新颖**：首次系统识别并形式化了CD-FSS中的语义过对齐和属性过对齐，并给出了针对性的层次化解法。
2. **方法设计巧妙**：利用槽注意力（slot attention）沿空间和通道做层次聚合，无需额外外部模型（如超像素分割），具有较好的通用性和可迁移性。
3. **在线概率语义库**：通过概率分布建模和采样提供多样化原型，避免在线微调的灾难性遗忘，且计算轻量。
4. **实验充分且结果领先**：在4个标准目标域上全面验证，并在1-shot和5-shot下均取得最优；与多种方法（包括使用Foundation model的方法）对比均胜出。
5. **效率高**：FLOPs和FPS均优于当前SOTA HSL，说明复杂度低。

## 8. 不足与局限
1. **边界精度有限**：与纯基于原型匹配的方法一样，在细粒度边界和细节区域分割精度不够高。作者在局限中承认，解码器类方法（如基于Transformer）边界更好但易过拟合，如何结合两者优势是未来方向。
2. **仅依赖单模态视觉信息**：未利用文本语义或跨模态先验，可能会限制对语义粒度的更准确理解。
3. **未见对极端跨域场景的测试**：如从自然图像到完全不同的医学CT/3D数据，当前实验覆盖了卫星、皮肤镜、X光、日常物体，但未覆盖视频或更复杂的医学模态。
4. **对训练策略的依赖**：使用全局风格随机化（GSR）和特定微调策略（每类first episode），是否对某些域不敏感需要进一步验证。
5. **未讨论OPSB对类别不均衡的敏感性**：如果某些类别在测试序列中出现较晚，早期构建的概率分布可能不准确。
