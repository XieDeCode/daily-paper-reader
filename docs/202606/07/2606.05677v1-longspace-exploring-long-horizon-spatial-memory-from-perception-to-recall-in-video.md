## 摘要
多模态大语言模型（MLLMs）在图像和视频理解方面取得了进展，并能够处理更长的视觉输入。自动驾驶和机器人导航等长程任务不仅需要识别当前视图，模型还必须记忆并检索先前观察到的空间布局、路径、视角变化和物体状态。为评估这一能力，我们引入了LongSpace-Bench，一个用于长程空间记忆的室内漫游视频基准，涵盖场景感知、空间关系和空间记忆。在本工作中，我们进一步提出了LongSpace，一个用于长视频空间推理的记忆框架。LongSpace将长视频建模为连续片段，将3D结构线索融入早期解码器层，并构建层级感知记忆以进行问题引导的检索。在多个空间推理基准上的实验表明，LongSpace提升了长视频空间理解能力，进一步证明了显式空间记忆是长程视频MLLMs的关键能力。

## Abstract
Multimodal Large Language Models (MLLMs) have advanced image and video understanding and can increasingly handle longer visual inputs. Long-horizon tasks such as autonomous driving and robotic navigation require more than recognizing the current view, as models must remember and retrieve previously observed spatial layouts, routes, viewpoint changes, and object states. To evaluate this capability, we introduce LongSpace-Bench, a room-tour video benchmark for long-horizon spatial memory, covering scene perception, spatial relations, and spatial memory. In this work, we further propose LongSpace, a memory framework for long-video spatial reasoning. LongSpace models long videos as sequential chunks, incorporates 3D structural cues into early decoder layers, and constructs layer-aware memory for question-guided retrieval. Experiments on multiple spatial reasoning benchmarks show that LongSpace improves long-video spatial understanding, further demonstrating explicit spatial memory as a key capability for long-horizon video MLLMs.

## 1. 核心问题与整体含义（研究动机和背景）

- **核心问题**：现有多模态大语言模型（MLLMs）在处理长视频时，**无法有效记忆和检索长时程中的空间信息**（如场景布局、路径、视角变化、物体状态）。自动驾驶、机器人导航等长时程任务需要模型不仅理解当前帧，还要回忆历史观察。
- **背景缺口**：现有空间基准多聚焦静态图像或短视频，缺乏对**长视频中跨片段的空间记忆**的评估。单纯增加输入帧数不足以解决问题，冗余视觉标记会冲淡重要空间线索，且无结构化的跨段信息难以检索。
- **本文目标**：提出新基准（LongSpace-Bench）和新框架（LongSpace），以评估和提升MLLMs在长视频中的空间推理与记忆能力。

## 2. 方法论：核心思想、关键技术细节

- **核心思想**：将长视频建模为连续片段（chunks），在解码器层注入3D结构线索，并构建**层级可检索的KV记忆**，实现“感知→记忆→检索→回答”的流程。
- **关键技术细节**：
  1. **空间结构感知（Spatial Structure Perception, SSP）**  
     - 使用3D几何编码器（π³）提取3D空间特征，与2D视觉特征对齐后，通过**低秩瓶颈（scale-shift + gate）**注入到**早期解码器层**的视觉token位置，仅需少量层（8层效果最佳）。
     - 公式：\( H_l^\Omega \leftarrow H_l^\Omega + \Delta_l(H_l^\Omega, G, q_l) \)，其中 \(\Delta_l\) 为几何调制的残差更新。
  2. **层级KV记忆（Hierarchical KV Memory, HKM）**  
     - 将视频按帧分块，每块经SSP编码后，在解码器各层按**角色（感知层/工作层/长期层）**选择重要KV状态并压缩存储。
     - 选择策略基于**显著性、变化、覆盖度、新近度**四个信号，压缩时对组内KV进行softmax加权池化，保留时序位置信息。
     - 推理时，用问题查询检索各层记忆：感知层和工作层进行稀疏top-k检索，长期层先匹配摘要再读取细粒度条目，将所有检索到的KV作为**冻结记忆前缀**注入解码器。
  3. **推理流程**（Algorithm 1）  
     - 分块编码并写入HKM → 编码问题 → 每层检索 → 生成答案。

## 6. 主要结论与发现

1. **LongSpace-9B在LongSpace-Bench总体得分49.2**，超过最强开源基线Qwen3-VL-32B（46.5）和专有模型Gemini-3-Pro（45.3），尤其领先空间中心模型（如Cambrian-S-7B 40.5）。
2. **显式空间记忆对记忆密集型任务增益最大**（如出现顺序、状态变化、路线回忆），而场景识别和高层规划仍受益于更强专有模型的先验。
3. 几何注入**仅需少量早期层即可生效**（8层最优），过多层反而下降。
4. **层级角色设计比单纯扩大记忆容量更重要**，删除长期记忆角色性能下降最显著。
5. 长视频下（>20分钟），记忆推理相比均匀采样提升15个百分点，证明跨段检索能力是关键。
