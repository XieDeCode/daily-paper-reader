## 摘要
视觉-语言-动作（VLA）模型在环境变化下（例如相机姿态变化或切换到不同但相似的机器人，如从Panda到UR5e）往往无法执行相同的已学习任务。将这些模型适应到变化后的环境（即目标域）通常需要每个任务在多个演示上进行训练，而收集这些演示成本高昂。为减轻数据收集和训练的负担，我们提出了一种基于类比的方法，通过权重向量算术和领域特定信息加法来适应环境变化下的VLA模型，命名为域算术（DART）。与之前的方法不同，DART只需收集单个演示，从而实现高效适应。为了精确地提取用于加法的领域特定信息，DART在权重向量的奇异分量之间进行子空间对齐，以滤除噪声分量。在模拟和真实世界实验中，DART在跨多种视觉和具身变化的单次场景中优于现有的VLA适应方法。代码可在https://github.com/snumprlab/dart获取。

## Abstract
Vision-Language-Action (VLA) models often fail to perform the same learned tasks under environmental shifts, such as changes in camera pose and shifts to a different but similar robot (e.g., from Panda to UR5e). Adapting these models to the shifted environment (i.e., target domain) often requires training on multiple demonstrations for each task, which are costly to collect. To reduce the burden of data curation and training, we propose an analogy-based method that adapts VLA models under environmental shifts through weight vector arithmetic with domain-specific information addition, named Domain ARiThmetic (DART). Unlike prior approaches, DART requires collecting only a single demonstration, enabling efficient adaptation. To accurately isolate domain-specific information for addition, DART performs subspace alignment between singular components in weight vectors to filter out noisy components. In both simulated and real-world experiments, DART outperforms existing VLA adaptation methods in one-shot scenarios across diverse visual and embodiment shifts. Code is available at https://github.com/snumprlab/dart.

### 1. 论文的核心问题与整体含义（研究动机和背景）
- **核心问题**：视觉-语言-动作（VLA）模型在训练环境（源域）中表现出色，但在环境发生改变（如相机位姿变化、机器人本体更换）的目标域中，性能会大幅下降。现有的适应方法通常需要针对每个任务收集多个专家演示数据进行微调，这在实际部署（如家庭环境）中成本高昂，难以规模化。
- **研究动机**：实现**单样本（one-shot）适应**，即仅利用目标域中**单个任务的一个演示**，就能让 VLA 模型适应整个目标域，并保持对源域学习到的多任务能力。
- **整体含义**：通过分析发现，单次微调产生的参数更新向量可近似分解为任务特定方向和领域特定方向，且二者可加性组合。基于此提出 DART 方法，通过权重算术（类比“皇后 = 国王 + 女人 - 男人”）提取领域向量，实现高效适应。

### 2. 论文提出的方法论
- **核心思想**：利用**类比操作**（weight arithmetic）从目标域和源域的单次微调权重中提取“领域向量”，将其加回基础模型，从而转移领域知识。
- **关键技术细节**：
  - **更新向量定义**：对于同一任务 \(T_m\)，在源域微调得到 \(\theta_{m,\text{src}}\)，在目标域微调得到 \(\theta_{m,\text{tgt}}\)，更新向量分别为 \(\Delta_{m,\text{src}} = \theta_{m,\text{src}} - \theta_0\) 和 \(\Delta_{m,\text{tgt}} = \theta_{m,\text{tgt}} - \theta_0\)。
  - **领域向量提取**（公式 3）：直接相减 \(\delta_{\text{tgt}} = \Delta_{m,\text{tgt}} - \Delta_{m,\text{src}}\)，以消除任务方向，保留领域方向。
  - **子空间过滤（Subspace Filtering）**：对源域更新向量进行 SVD 分解，仅保留与目标域子空间对齐的基向量（通过能量阈值 \(\gamma^{(l)}\) 控制），滤除源域噪声。
  - **子空间缩放（Subspace Scaling）**：用子空间对齐分数 \(\gamma^{(l)}\) 缩放领域向量，抑制低对齐、噪声主导的分量。
  - **最终适应**（公式 7）：\(\theta^* = \theta_0 + \alpha \cdot \tilde{\delta}_{\text{tgt}}\)，其中 \(\alpha\) 是缩放系数。
- **算法流程**：1）对同一任务 \(T_m\) 在源域和目标域分别进行单次演示微调（1000步）；2）计算更新向量；3）对每一层进行 SVD、子空间对齐、过滤和缩放，得到精炼领域向量；4）将领域向量乘以系数 \(\alpha\) 加回基础模型。

### 6. 论文的主要结论与发现
- **主要结论**：DART 在单样本适应环境下显著优于所有基线，在 LIBERO 上平均成功率比次优方法 FLA 高约 5-6 个百分点（Medium 视角 80.8% vs 76.4%），在真实世界高 26.7 个百分点（81.7% vs 55.0%）。
- **关键发现**：
  - 单次微调权重中，任务方向和领域方向可加性分解且近似正交。
  - 从领域向量中通过子空间对齐能有效去除噪声，提升适应质量。
  - DART 方法对超参数 α 鲁棒，且适应后不会忘记源域能力。
  - 多个领域的领域向量可合并为一个向量，同时适应多个目标域。

### 7. 优点
- **数据极度高效**：仅需目标域单个任务一个演示，即可完成整个环境的适应。
- **架构无关**：在 π0.5 和 π0-FAST 上均有效，不依赖特定模型结构。
- **无需修改模型架构**：仅通过权重算术操作，即插即用。
- **稳定且鲁棒**：对超参数 α 不敏感，即使微调步数不足也能保持良好性能。
- **可合并性**：领域向量可合并，降低了多域部署的存储开销。
- **不遗忘源域**：适应后未出现灾难性遗忘（源域性能几乎不变）。

### 8. 不足与局限
- **严重漂移下性能局限**：在 Large 视角等极端漂移下，所有单样本方法性能都较低，DART 虽优于基线但仍有提升空间。
- **需要超参数 α**：虽然鲁棒，但仍需少量（10次 rollout）搜索最佳 α，无法完全免调参。
- **需要源域演示**：要求同一任务在源域有一个可用演示（通常可从训练集获取），但若任务不匹配则性能下降（虽可通过相似任务检索缓解）。
- **未讨论更复杂的漂移**：如动态光照变化、传感器噪声、物体外观变化等未充分覆盖。
- **实验未涉及更大规模 VLA 模型或更多机器人平台**：仅在 π0.5 和 π0-FAST 上验证，对更大模型（如 π0）未测试。
