## 摘要
树冠对于韧性人工智能是一个具有挑战性的目标，因为它们不是静态物体：其光谱响应、内部纹理、半透明性和表观边界在生长季节中会显著变化。我们开发了PhenoEmbed，一个自监督的以树冠为中心的时间序列嵌入模型，该模型在HeideBench（Döllauer Heide森林树冠物候的18期无人机多光谱时间序列基准）上使用对比损失和掩码重建损失进行训练。该模型将季节性树冠动态视为由叶片萌发、林冠闭合、衰老和落叶条件驱动的物候外观变化。保留分割后的树冠多边形作为对象锚点，以提取随时间对齐的以树冠为中心的小块，从而为每棵树学习一个256维向量来总结季节性树冠外观。在5885个无损坏的树冠上，导出的嵌入显示出结构化的低维组织，前两个主成分解释了25.1%的方差，最近邻检索的中位top-1余弦相似度为0.946。与手工制作的时间特征和学习的平均池化基线相比，PhenoEmbed产生了更紧凑的最近邻结构，而消融实验表明，对比损失、掩码重建损失和明确的季节性时间特征各自影响所学嵌入空间的结构。这些结果支持PhenoEmbed作为一种可复用的森林树冠表示学习器，并激发未来的下游测试，以探究这些特征是否能在季节性变化下改进树木级模型。

## Abstract
Tree crowns are a challenging target for resilient AI because they are not static objects: their spectral response, internal texture, translucency, and apparent boundaries change substantially across the growing season. We develop PhenoEmbed, a self-supervised crown-centric temporal embedding model trained with contrastive and masked reconstruction objectives on HeideBench, an 18-date UAV multispectral time-series benchmark for forest crown phenology in D{ö}lauer Heide. The model treats seasonal crown dynamics as phenological appearance change driven by leaf emergence, canopy closure, senescence, and leaf-off conditions. Segmented tree crown polygons are retained as object anchors to extract aligned crown-centered crops through time, allowing one 256-dimensional vector summarizing seasonal crown appearance to be learned per tree. On 5,885 crop-safe crowns, the exported embeddings show structured low-dimensional organization, with the first two principal components explaining 25.1\% of variance and nearest-neighbor retrieval producing a median top-1 cosine similarity of 0.946. Compared with handcrafted temporal features and a learned mean-pooling baseline, PhenoEmbed yields substantially more compact nearest-neighbor structure, while ablations show that the contrastive loss, masked reconstruction loss, and explicit seasonal time features each affect the structure of the learned embedding space. These results support PhenoEmbed as a reusable forest crown representation learner and motivate future downstream tests of whether such features improve tree-level models under seasonal change.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题背景**：树冠并非静态对象，其光谱响应、内部纹理、半透明性和表观边界在生长季节中会发生显著变化（从叶片萌发、林冠闭合、衰老到落叶）。这种季节动态导致基于单时间点的分割或分析模型在不同物候阶段变得脆弱，是地球观测中一个典型的韧性AI问题。
- **核心动机**：传统范式是“树标注 + 航空/UAV栅格 = 分割”，但当目标图像与训练图像处于不同物候状态时效果不佳。本文希望建立“树标注 + 航空/UAV栅格 + 物候嵌入 = 更韧性的分割”的新范式，即学习一种能够显式捕捉季节性动态的树冠表示（嵌入），以便下游模型在季节变化下仍能鲁棒工作。
- **整体含义**：PhenoEmbed 是首个面向 UAV 多光谱时间序列的、以树冠对象为中心的、自监督的物候嵌入方法，旨在提供一个可复用的森林树冠特征学习器，有望减少未来需要在不同季节重新训练下游模型的必要性。

## 2. 方法论

### 核心思想
- 利用已有的分割树冠多边形作为对象锚点，从多时相多光谱正射影像中提取对齐的以树冠为中心的裁剪块（crops），通过自监督学习（对比学习 + 掩码重建）为每个树冠学习一个 256 维的嵌入向量，该向量总结整个生长季的树冠外观变化。
- 将季节树冠动态视为物候外观变化，而非噪声或干扰加以抑制。

### 关键技术细节
1. **数据准备**：
   - 使用 HeideBench 基准数据集（18 期 UAV 多光谱正射影像，2025-03-06 至 2025-11-05，每期 4 个波段：R、G、RE、NIR）。
   - 从 DeepTrees 库获得 5885 个“作物安全（crop-safe）”的树冠多边形（完全位于所有 18 期影像有效区域内的树冠）。
   - 为每个树冠在每个日期提取 16m × 16m 平方裁剪块（288×288 像素），保存为 4 波段张量。
2. **空间编码器**：使用 SegFormer MiT-B2 作为共享的逐日期空间编码器。由于预训练骨干期望 3 通道输入，添加 1×1 可训练光谱适配器将 4 通道映射到 3 通道（初始化为物理驱动：R→红，G→绿，RE 和 NIR 均值→蓝）。
   - 骨干冻结，仅训练适配器和时间头部。
3. **季节时间编码**：将每个日期的归一化季节坐标 τ ∈ [0,1] 编码为 4 维特征向量 [τ, τ², sin(2πτ), cos(2πτ)]，通过线性投影加到 Transformer 的输入标记上。
4. **时间聚合**：采用 2 层 Transformer（4 头，隐层维度 256）处理逐日期特征序列，并加上一个可学习的类别标记（CLS token）。最终将 CLS token 输出经线性投影和 L2 归一化得到最终的 256 维树冠嵌入。
5. **自监督目标**：联合两个损失：
   - **时间对比损失**（InfoNCE 风格）：对同一树冠的两个独立掩码视图（随机掩码约 30% 的日期）产生的嵌入拉近，对不同树冠推开。批次大小设为 2（基线），并实验了批次大小 16 的敏感性。
   - **掩码重建损失**：预测被掩码日期的每个波段的裁剪块均值（4 维向量），使用 MSE 损失。
   - 总损失：L = λ_ctr * L_ctr + λ_rec * L_rec，两个权重均为 1.0。
6. **训练细节**：AdamW 优化器，学习率 5e-5，权重衰减 1e-4，最大 50000 步，早停 patience 20 次验证检查。使用 PyTorch Lightning 管理。

## 6. 主要结论与发现

1. **季节信号强**：手工提取的多边形池化轨迹显示不同树冠具有不同幅度和时序的物候信号，证明物候建模合理。
2. **嵌入质量好**：

3. **消融揭示贡献**：
   - 去除对比损失或重建损失仍可得到紧凑嵌入，但每个目标均塑造了空间结构。

5. **主要贡献**：提出了一种可复用的森林树冠表示学习器，证明了对象中心时间嵌入的可行性。
