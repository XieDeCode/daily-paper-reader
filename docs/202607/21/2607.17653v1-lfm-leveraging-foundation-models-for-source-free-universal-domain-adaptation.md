## 摘要
源无关通用域自适应（SF-UniDA）旨在将预训练的源模型适应到未标注的目标域，同时处理协变量偏移和标签偏移，且无法访问源数据。然而，现有的SF-UniDA方法依赖于阈值调整和聚类等低效技术。基础模型（FMs）以其泛化性和零样本能力著称，但在SF-UniDA中尚未得到充分探索。本文提出了一种利用基础模型（LFM）的SF-UniDA框架。我们使用视觉-语言模型（VLM）计算目标样本与文本标签（包括通过提示大语言模型生成的未知类别标签）之间的相似度。通过分析基于相似度的样本级得分的变异系数，确定标签偏移类型。利用拟合于另一个基于相似度度量的二元高斯混合模型来识别未知样本。在共识策略下，由VLM生成的伪标签通过初始化为预训练源模型的目标模型进行精炼，整合来自源领域和基础模型的知识。最后，这些精炼后的伪标签用于训练目标模型。在所有可能的标签偏移和多个基准上的大量实验证明了我们提出的LFM框架的有效性和优越性。我们的代码可在https://github.com/iamjingli/LFM获取。

## Abstract
Source-free universal domain adaptation (SF-UniDA) adapts a pre-trained source model to an unlabeled target domain under both covariate and label shifts, without access to source data. However, existing SF-UniDA methods rely on inefficient techniques such as threshold tuning and clustering. Foundation models (FMs), known for their generalization and zero-shot capabilities, remain underexplored in SF-UniDA. In this paper, we propose a framework that leverages foundation models (LFM) for SF-UniDA. We use a vision-language model (VLM) to compute similarities between target samples and text labels, including those for unknown classes generated by prompting a large language model. The label shift type is determined by analyzing the coefficient of variation of a similarity-based sample-level score. Unknown samples are identified using a binary Gaussian mixture model fitted to another similarity-based metric. Under a consensus strategy, the pseudo-labels generated by the VLM are refined by the target model initialized with the pre-trained source model, integrating knowledge from both the source domain and foundation models. Finally, these refined pseudo-labels are used to train the target model. Extensive experiments across all possible label shifts and multiple benchmarks demonstrate the effectiveness and superiority of our proposed LFM framework. Our code is available at https://github.com/iamjingli/LFM.

## 1. 核心问题与整体含义（研究动机和背景）

- **问题定义**：源无关通用域自适应（SF-UniDA）要求在无法访问源数据的情况下，将预训练的源模型适应到未标注的目标域，同时处理两类偏移——协变量偏移（数据分布不同）和标签偏移（目标域可能包含比源域更多或更少的类别）。标签偏移包括三种类型：**部分集偏移（PDA）**（目标类别是源子集）、**开放集偏移（OSDA）**（目标包含源未知类别）、**开放部分集偏移（OPDA）**（两类都有未知类别）。由于目标域无标注，事先无法知道偏移类型。
- **现有方法的不足**：传统UniDA需要源数据；源无关方法中，要么依赖手工阈值调整（不鲁棒），要么依赖聚类（计算昂贵且对高维数据不稳定）。基础模型（如CLIP、LLM）的零样本能力和泛化性尚未被系统引入SF-UniDA。
- **本文贡献**：提出**LFM**框架，首次系统性地利用视觉-语言模型（VLM）和大语言模型（LLM）作为先验知识，结合源模型的知识，实现无源情况下对所有标签偏移类型的自适应，同时避免手动阈值和聚类。

## 2. 方法论

### 核心思想
- 使用**LLM**生成潜在未知类别的文本标签；使用**VLM（CLIP）**计算目标图像与所有类别（已知+未知）文本的相似度，基于这些相似度设计样本级得分，进而判断标签偏移类型、分离未知样本，并生成初始伪标签；最后通过**共识策略**用目标模型（源自源模型）精炼伪标签，融合双方知识。

### 关键技术细节

#### 2.1 未知类标签生成（LLM）
- 给定已知类别标签集合，利用GPT-4-turbo（具体版本）生成数目为 `C_unk = α * C_s` 的潜在未知类标签（α=2）。
- 提示模板参考EOE方法，要求LLM基于视觉相似性生成与已知类别“视觉相似但无关”的类别。
- 输出后进行重叠检查，若与已知类重叠则重新生成。

#### 2.2 样本级得分定义
- CLIP计算目标图像 `I_i` 与文本 `T_j` 的相似度 `Sim_ij`，缩放后 `aSim_ij = Sim_ij / t`（t=0.01）。
- **score_i**：`score_i = max_{j∈[1,Cs]} softmax_j - β * max_{k∈(Cs,Cn]} softmax_k`，用于检测标签偏移（β有数据集差异，Office-31/DomainNet为0.25，Office-Home/VisDA为0.75）。
- **rate_i**：`rate_i = max_{j∈[1,Cs]} Sim_ij - β * max_{k∈(Cs,Cn]} Sim_k`，用于GMM分离未知样本（避免score_i过度极化导致的误分）。

#### 2.3 标签偏移类型判定（LSTD）
- 计算所有目标样本score_i的变异系数 `cv = std(score) / mean(score)`，仅用top 30%和bottom 30%的得分。
- 若 `cv > δ`（δ=1.0），判定为OSDA或OPDA（存在未知类）；否则为PDA。

#### 2.4 未知样本分离（GMM）
- 若为OSDA/OPDA，对rate_i拟合**二元高斯混合模型（GMM）**，低分簇视为未知样本，赋予“unknown”标签；高分簇视为已知样本。
- 该操作避免了手动设置阈值。

#### 2.5 伪标签生成与精炼（PLR）
- 已知样本：CLIP生成初始伪标签 `y_clip_i`（基于已知类softmax最大值），并保留top-3候选集 `P_i`。
- 目标模型（由源模型初始化）对每个样本输出概率向量`p_i`，通过K近邻（K=5）平均得到`¯p_i`，生成伪标签 `y_model_i`。
- **共识精炼规则**：  
  `y_hat_i = y_model_i` 若 `(conf_model_i > conf_clip_i) AND (y_model_i ∈ P_i)`，否则保留 `y_clip_i`。
- 该规则要求模型置信度更高且预测在CLIP的top-3候选集中，才替换伪标签。

#### 2.6 训练损失
- 已知样本：交叉熵损失 `L_k`。
- 未知样本：自熵最大化损失 `L_unk`（鼓励对未知类的均匀预测）。
- 总损失 `L = L_k + L_unk`。
- 推理时：使用目标模型预测，基于归一化熵阈值w=0.55判断是否拒绝（输出未知）。
