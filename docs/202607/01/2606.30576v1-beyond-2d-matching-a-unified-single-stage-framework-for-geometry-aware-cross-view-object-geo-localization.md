## 摘要
跨视角目标地理定位（CVOGL）旨在从查询视图（例如地面或无人机）中定位目标对象，并在带有地理标签的参考图像（例如卫星）中定位。现有方法严重依赖2D外观匹配，并且受限于缺乏几何元数据、多样化提示和标准视场图像的有限数据集。为了解决这些相互交织的挑战，我们首先引入了\dataset，一个大规模、高保真的建筑数据集，包含超过22万对地面-卫星和无人机-卫星图像。它提供了多模态提示（点、框、掩码）和相机姿态，以实现灵活的目标引用和显式空间建模。此外，我们提出了一种新颖的单阶段几何感知地理定位框架（GAGeo），该框架基于置换等变3D基础模型$π^3$。通过无缝集成视觉特征、提示和可学习任务令牌，我们的模型在单次前向传播中适应继承的3D先验，联合预测边界框、分割掩码和相机姿态。此外，我们引入了一种对比损失，利用卫星视图作为通用锚点，隐式对齐地面和无人机表示，从而实现零样本地面到无人机的定位，无需三元组训练数据。大量实验表明，我们的方法显著优于最先进的方法，在未见场景和新颖的跨视角设置中展现出卓越的泛化能力。

## Abstract
Cross-view object geo-localization (CVOGL) aims to locate a target object from a query view (e.g., ground or drone) within a geo-tagged reference image (e.g., satellite). Existing approaches heavily rely on 2D appearance matching and are constrained by limited datasets lacking geometric metadata, diverse prompts, and standard field-of-view imagery. To address these intertwined challenges, we first introduce \dataset, a large-scale, high-fidelity building dataset comprising over 220,000 ground-satellite and drone-satellite pairs. It provides multi-modal prompts (points, boxes, masks) and camera poses to enable flexible target referring and explicit spatial modeling. Furthermore, we propose a novel single-stage Geometry-Aware Geo-localization framework (GAGeo), built upon the permutation-equivariant 3D foundation model $π^3$. By seamlessly integrating visual features, referring prompts, and learnable task tokens, our model adapts the inherited 3D prior to jointly predict bounding boxes, segmentation masks, and camera poses in a single forward pass. Additionally, we introduce a contrastive loss that utilizes the satellite view as a universal anchor, implicitly aligning ground and drone representations to enable zero-shot ground-to-drone localization without requiring triplet training data. Extensive experiments demonstrate that our approach significantly outperforms state-of-the-art methods, exhibiting exceptional generalization ability in unseen scenes and novel cross-view setups.

## 2. 论文提出的方法论
### 核心思想
- **数据层面**：构建了**CMA-Loc**数据集，包含超22万对地面-卫星和无人机-卫星实例对，覆盖8个全球城市，提供多模态提示（点、框、掩码）和精确相机姿态（位置+旋转），并包含手动标注的地面-无人机-卫星三元组用于零样本评估。

### 关键技术细节
- **图像与提示编码**：使用冻结的DINOv2提取图像token；SAM2预训练模块处理多模态提示（点/框→稀疏token，掩码→密集嵌入）。
- **Prompt注入与跨视角交互**：
  - 在参考流（卫星）中添加可学习任务token（包括目标token和位置token）。
  - 稀疏提示token直接拼接至查询流；密集掩码嵌入在全局注意力中作为键的调制信号，不改变输出特征分布。
  - 全局注意力中使用方向隔离掩码：提示token只能关注查询token，与参考流隔离。
- **任务解码**：
  - 目标token经DETR-style MLP输出边界框，经SAM-style超网络生成分割掩码。
  - 位置token与上采样参考特征点积生成热图预测相机像素位置。
  - 相机头（继承π³）预测相机旋转（欧拉角/旋转矩阵）。
- **优化目标**：联合损失函数 \( L_{total} = \sum_{k\in K} w_k(L_{grd}^{(k)}+L_{pos}^{(k)}+L_{rot}^{(k)}) + L_{cl} \)。
  - \( L_{grd} \)：匈牙利匹配后，分类（Focal loss）+ 边界框回归（L1+GIoU）+ 掩码（BCE+Dice）。
  - \( L_{pos} \)：对热力图使用Focal loss。
  - \( L_{rot} \)：旋转矩阵的测地线误差。
  - \( L_{cl} \)：对比损失，以卫星掩码池化特征为锚点，拉近其与正例查询特征的距离，推开负例（MoCo队列）。

## 6. 主要结论与发现
- **GAGeo在所有设置下显著超越现有SOTA**。例如：
  - 地面→卫星检测mAcc提升约34%（从~46%到~71%），分割mIoU提升约33%（从~33%到~72%）。
  - 零样本地面→无人机检测mAcc达43.86%，分割mIoU达57.88%，较之前最好方法TROGeo提升约20个百分点。
- **利用3D几何先验有效克服2D匹配局限**，尤其处理极端视角变化和重复纹理。
- **对比损失使卫星作为桥接**实现零样本跨视图泛化，无需三元组训练数据。
- 多模态提示（框/掩码）优于仅点提示，且单任务token即为最优（过多token引入噪声）。

## 7. 优点
- **创新性方法**：首次将3D基础模型π³引入CVOGL，利用其置换等变性和共享表征空间，统一处理地面→卫星和无人机→卫星。
- **单阶段架构**：摒弃传统两阶段（检测+分割）或独立解码头，在单次前向中同时输出多任务结果，简洁高效。
- **数据贡献**：CMA-Loc数据集规模大（18倍于之前）、包含多模态提示和相机姿态、标准FOV图像、全球多样建筑风格，推动领域进步。
- **强大泛化**：在未见场景、未见数据集、以及新颖的跨视图组合（地面→无人机）上均表现出色，验证了模型的学习到的是几何不变先验而非过拟合。
- **严谨消融**：详细分析了损失函数、token数量、骨干选择的影响。

## 8. 不足与局限
- **推理效率**：框架当前受限于3DFM骨干的架构复杂性（π³有18层，但对比VGGT 24层更轻量），作者承认这是主要瓶颈，未来可考虑token合并、剪枝或知识蒸馏。
- **计算资源未披露**：缺少实际训练所需的算力描述，难以评估可复现性和资源门槛。
- **单目标局限**：当前设计针对单目标定位，多目标场景需要扩展（如增加可学习token数量并设计匹配策略）。
- **泛化边界未被系统探索**：仅测试了城市建筑场景（8个城市），对非城市环境（乡村、森林）或非刚性目标（车辆、行人）尚未验证。
- **对比实验未包含最新3D FM方法**：未与基于Mast3R或DUSt3R的细粒度定位方法直接对比（由于任务定义差异，但可增加讨论）。
